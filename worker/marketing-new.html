encryption

vault

aes-256-gcm local encryption. secrets encrypted on your machine, never transmitted. VAULT: prefix for auto-decrypt at runtime.

how it works

generate a local key, encrypt your secrets, commit the encrypted values safely. gate decrypts VAULT:-prefixed values automatically at runtime.

  • aes-256-gcm authenticated encryption
  • keys generated and stored locally
  • encrypt individual values or entire .env files
  • VAULT: prefix for auto-decryption
  • safe to commit encrypted values to git
  • zero network dependency
vault — encrypt
